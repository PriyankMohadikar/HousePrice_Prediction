# 🏠 House Price Prediction (ML Project)

## 📌 Overview
This project predicts house prices using Machine Learning models like Linear Regression, Ridge, and Lasso.

## 🔧 Features Used
- OverallQual
- GrLivArea
- GarageCars
- TotalBsmtSF
- FullBath
- HouseAge
- RemodelAge

## ⚙️ Models Used
- Linear Regression
- Ridge Regression (Tuned)
- Lasso Regression (Tuned)

## 📊 Results
- Best R² Score: ~0.84
- MAE: ~$20,000

## 🚀 Steps
1. Data Cleaning & Feature Engineering
2. Log Transformation
3. Model Training & Evaluation
4. Hyperparameter Tuning
5. Final Prediction
